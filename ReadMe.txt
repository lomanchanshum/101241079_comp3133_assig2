Employee Management App

A brief description of the project.
Installation

To run the app, you must install the following software:

    -JDK 17 or the latest version
    -Java Eclipse IDE
    -Maven
    -MongoDB
    -Node.js
   -AngularJS
   -NPM packages

Usage

To run the app, follow these steps:

    -Start MongoDB by running the following command in the terminal:
    mongod

   -Start the backend server by running the following command in the terminal:
    mvn spring-boot:run

    -Start the frontend server by running the following command in the terminal:
    npm start

    -Navigate to http://localhost:4200 in your web browser to access the app.
